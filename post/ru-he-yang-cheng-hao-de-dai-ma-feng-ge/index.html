如何养成好的代码风格？

2020-02-27

代码风格这个事情是让人又爱又恨的。

因为，代码风格对于不同的人来说有着不同的定义。

举个栗子：

//A代码：
for(int i=1;i<=n;i++){
    ans+=i;
}

// B代码：
for ( int i = 1 ; i <= n ; i++){
    ans += i;
}

//C代码：
int i;
for(i=1;i<=n;i++)
{
    ans=ans+i;
}

相信许多人的写法都是不一样的。

那，怎么养成好的代码风格呢？

此处建议仅为个人平日练习代码风格

  1. 命名规则。
    要么都选大驼峰 getMap ，小驼峰 getmap ，分割法 get_nap,等等。
