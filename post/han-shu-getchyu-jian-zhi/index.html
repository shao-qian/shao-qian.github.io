【函数】getch()与键值

2020-02-27
  • getch()
    getch()在conio.h库内,使用时请在头文件加上#include<conio.h>

getch()用于获取键值。

  • 键值
    键盘上每按下一个键，会被getch()监听到，返回一个值。

这个值就叫键值。

关于每个键对应的键值，请看如下程序：

#include<bits/stdc++.h>
#include<conio.h>
using namespace std;
int main(){
    while(1){
        cout<<(int)getch()<<endl<<"------"<<endl; 
    }
    return 0;
}

运行时，每次按下一个键就会输出一个数。

这就是这个键的键值。

特别的，当你按下上下左右键时，会返回两个数，一个是224，另一个是它的键值。

  • 应用

对键值的判断，可以用来看按下了哪个键。

请看如下程序：

#include<bits/stdc++.h>
#include<conio.h>
using namespace std;
int main(){
    while(1){
        int n=getch();
        if(n==72){
            cout<<"上"<<endl;
        }
    }
    return 0;
}
